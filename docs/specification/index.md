# IEC63430 とは

IEC 63430 で国際規格化が進められている「コンテナフォーマット」は、ウェアラブルセンサ信号に係る互換性や 5G システムと連携する IoT への適応性を高め、データの共有・利活用を柔軟に行える、標準化の核となる技術です。

本規格は IEC TC100/TA18/PT63430 にて議論がすすめられています。
